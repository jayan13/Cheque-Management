Cheque Management Documentation

Detailed explanation for Cheque Management application usage

Cheque Management is used to register all receivable and payable Cheques

Cheque Management is based on ERPNext done using Frappe Framework.

Cheque Management is Open Source under the GNU General Public Licence v3.

Install

Using Bench

Go to your bench folder and setup the new app

$ bench get-app cheque_management https://github.com/frappe/erpnext
$ bench new-site testsite
$ bench --site testsite install-app cheque_management

Login to your site to use the app.
